Avtars

Start using these reusable avtars in your projects!

You can use img-tp-2 (top), img-btm-2(bottom), img-rt-2(right) img-lt-2(left) to the img element for aligning images, numbers in those classes represent rem. you can give values of 2, 4 and 6 rem

Container width and heights are same thus only one class for those img height and width classes are as follows img-h-6 img-w-8 the number in it defines rem. you can give values of 2, 4 and 6 rem
